CompareItemsProc

Definition

typedef Int8 (*CompareItemsProc)(void* item1, void* item2);

Description

Compares two items in a list.

Arguments

item1

The first item to compare with the second.

item2

The second item to compare with the first.

Return

The callback should return:

  • -1 if item1 < item2
  • 0 if item1 == item2
  • +1 if item1 > item2
