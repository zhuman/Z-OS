Lists

Z-OS provides a comprehensive list library for use both within the system as well as for the convenience of users. A list is a simple linked-list of pointer-sized data. Lists can therefore store a collection of allocated structures by pointer. Note that none of the list functions are thread-safe, so semaphores and other synchronization primitives must be used to ensure that only one thread accesses a list at one time.

Lists are not objects, and so have a set of static functions:

  • GetListItem
  • AddListItem
  • RemoveListItem
  • ClearList
  • ReplaceListItem
  • InsertListItem
  • SwapListItems
  • SortList
  • GetIndexOf
  • ConcatenateLists
