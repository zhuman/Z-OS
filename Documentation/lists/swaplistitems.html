SwapListItems

Definition

void SwapListItems(List* list, UInt16 item1, UInt16 item2);

Description

Swaps two items in a list.

Arguments

list

A pointer to a List structure that forms the basis for this list.

item1

The index of the first item to swap.

item2

The index of the second item to swap.
