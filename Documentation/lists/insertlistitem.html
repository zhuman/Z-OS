InsertListItem

Definition

void InsertListItem(List* list, UInt16 index, void* newItem);

Description

Inserts an item into a list at a specified index.

Arguments

list

A pointer to a List structure that forms the basis for this list.

index

The index at which to insert the item. This is now the item's index.

newItem

The new item to add to the list.
