FinishWait

Definition

Int16 FinishWait(void* data, ThreadInternal* specThread);

Description

This function is called by type managers once an object releases one or all of the threads waiting upon it. User code should never call this.

Arguments

data

The data field of the object that is being waited upon. This is used to identify the object.

specThread

A pointer to the ThreadInternal structure for a thread that is currently waiting on the object. If this is null, all threads that are waiting for the specified object will be released from their waiting state. If this is not null, only this specific thread will be released. This is used, for example, to only allow a set number of threads access to a semaphore at one time.

Return

Error Code
