WaitForObject

Definition

Int16 WaitForObject(UInt16 handle);

Description

This method suspends execution of the current thread until the object waited upon signals to resume. For example, waiting on a timer suspends the thread until the timer has timed out. See WaitForObjects for waiting upon multiple objects.

Arguments

handle

The handle to the object who is to be waited upon.

Return

Error Code
