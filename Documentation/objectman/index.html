The Object Manager

Z-OS's entire API minus the Object Manager itself is accessible through a system of objects and interfaces. If this is a new concept, you should read over the Basic Concepts of Z-OS. Each object can implement any number of interfaces, and the interfaces allow one to actually interact with the object.

In order to create and release objects, and receive interfaces for them, the object manager provides a small static API. It contains the following methods:

  • CreateObject
  • GetInterface
  • ReleaseObject
  • WaitForObject
  • WaitForObjects
  • FinishWait
  • IsHandleValid
  • RegisterTypeManager
  • CreateSymbolicLink
  • ChangeSymbolicLink
  • InternalObjectFromName
  • InternalObjectFromHandle
  • InternalObjectFromData
