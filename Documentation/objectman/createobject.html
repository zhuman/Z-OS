CreateObject

Definition

Int16 CreateObject(UInt16 type, UInt16* handle, char* name);

Description

This method creates a new object of a specified type and generates a new handle for it. Interfaces can be retrieved from the object by the GetInterface method. It can be released when no longer in use by calling ReleaseObject and passing the handle received from this function.

Arguments

type

The type code for the new object.

handle

A pointer to a variable that, when the function returns, will contain the handle of the object.

name

The null-terminated string that is to be the object name or null if the object should not have a name.

Return

Error Code
