ReleaseObject

Definition

Int16 ReleaseObject(UInt16 handle);

Description

This method releases a handle to an object once it is no longer in use by the thread. It is expected that all threads will release their handles before terminating as to not suffer a memory leak. Once an unnamed object has no open handles, it will be destroyed by the object manager.

Arguments

handle

The handle to release.

Return

Error Code
