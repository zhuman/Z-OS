IWaitable.StartWait

Definition

Int16 (*StartWait)(UInt16 handle);

Description

This function is implemented by type managers to provide the ability for threads to wait for an object. Internally, this function is called by the wait functions and is used to signal the start of a wait to type managers. It also allows type managers to signal that no wait is required and to immediately return from the wait function without a wait.

Arguments

handle

The handle to the object that is being waited upon.

Return

Error Code

Note that if this function returns anything other than ErrorSuccess, the wait function will immediately return, assuming that no wait was required or that an error occurred in registering the wait. Type manager should return ErrorNoWait when no wait is needed but no actual error occurred.
