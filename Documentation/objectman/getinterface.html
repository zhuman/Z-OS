GetInterface

Definition

Int16 GetInterface(UInt16 handle, UInt16 code, void** interface);

Description

This method attempts to retrieve an interface for a given object. If successful, the interface returned will contain functions that can be used to manipulate the object.

Arguments

handle

A handle to the object for whom to retrieve the interface.

code

The interface code of the interface to be retrieved.

name

The null-terminated string that is to be the object name or null if the object should not have a name.

Return

Error Code
