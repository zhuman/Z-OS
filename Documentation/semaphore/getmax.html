ISemaphore.GetMax

Definition

UInt16 (*GetMax)(UInt16 handle);

Description

Returns the maximum capture count of this semaphore object. Once the semaphore has been captured this many times, other threads must wait for it to be released by one or more other threads.

Arguments

handle

The handle to the object who's maximum capture count is to be returned.

Return

The maximum capture count for the object.
