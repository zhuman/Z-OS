Semaphores

Semaphores provide a simple method of inter-thread synchronization. A semaphore can be captured by a set number of threads only. Once it has reached its maximum capture count, other threads must wait their turn to capture it. A semaphore with its maximum thread capture count set to 1 is also known as a mutex (that is, mutual exclusion). This is the default setting for semaphores.

Implemented Interfaces

  • IGeneric
  • IWaitable
  • ISemaphore (see below)
