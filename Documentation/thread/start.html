IThread.Start

Definition

UInt16 (*Start)(UInt16 handle, ThreadFunction startFunc, void* arg);

Description

This method starts a thread object at a given function and allows the passing of an argument. Note that the thread is not immediately switched to. Rather, it is queued by the Thread Manager and will begin executing when it gets its turn.

Arguments

handle

A handle of the thread object to start.

startFunc

A pointer to the function to run in the thread.

arg

An argument passed to the function of the thread.

Return

Error Code
