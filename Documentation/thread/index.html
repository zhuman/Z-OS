Threads

Thread objects provide fine control over starting, stopping, suspending, and managing threads. The IThread interface acts as the Thread Manager's primary API for thread management.

Implemented Interfaces

  • IGeneric
  • IThread (see below)

IThread Methods

  • SetPriority
  • GetPriority
  • Start
  • GetState
  • Suspend
  • Resume
  • Stop
