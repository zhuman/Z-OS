Error Codes

Virtually all Z-OS APIs return a standard error code. Error codes are always of the type Int16, that is, a signed 16-bit integer. Definitions for all of the possible error codes can be found in Error.h. Codes are split into sections with each section providing all of the error codes for a particular subsystem of Z-OS. As a general rule, each section is allowed a 0x100 chunk of codes.

If a method is successful, it should return ErrorSuccess, which is notably zero.
