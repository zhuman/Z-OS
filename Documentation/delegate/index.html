Delegates

Delegates are an object-level representation of function pointers. A delegate internally stores a listing of function pointers called "handlers." Handlers can be added or removed by any thread, since delegates are thread-safe. When a delegate is invoked, all of its handlers are called. This allows for event-based programming. Delegates can be created by passing TypeDelegate as the type code to CreateObject.

Delegates also provide access to the system's worker thread pool. The worker thread pool is a group of special system-managed threads that run work items when requested. This allows functions t be called asynchronously without having to create new threads. The work load is automatically balanced amongst the various worker threads.
