Z-OS Documentation

Z-OS is a preemptively multitasking real-time operating system environment. It was designed with multiple goals in mind:

  • It must run flawlessly on current PIC24H devices (16-bit microprocessors from Microchip)
  • A highly modern object-oriented and unified API for use throughout the system
  • It should be highly encapsulated and easy to maintain
  • Adaptable to any workload
  • Provides all basic operating system and advanced runtime (e.g. lists, buffers, queues, etc.) functions out-of-the-box
  • Be easy to configure at compile-time
  • Can function as a "black box," that is, all configurations etc. are done outside of the source files
  • Should be easily portable to the new PIC32MX platform when needed
  • The user should never have to resort to disabling interrupts
  • The API should be logical and, being object-oriented, easily extensible by both the user and the developers

It can be believed that all of these goals have been achieved. This documentation library can be used both as a reference when developing and as a basic introduction to the system. If this is your first time using Z-OS, you probably want to check out the Basic Concepts section before preceding.
