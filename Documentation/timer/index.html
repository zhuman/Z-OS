Timers

Timers are an important part of any time-dependant operation, especially in a real-time operating system. Timers can be created by passing TypeTimer to CreateObject as the type code.

Implemented Interfaces

  • IGeneric
  • IWaitable
  • ITimer (see below)

ITimer Methods

  • SetInterval
  • GetInterval
  • Start
  • Stop
  • Reset
  • IsRunning
