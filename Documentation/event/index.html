Events

Events provide a simple method of synchronization between threads. An event can be in one of two states, "set" and off. Waiting threads are released whenever the event is in the "set" state. Events can be created with CreateObject, passing TypeEvent as the type code.

Implemented Interfaces

  • IGeneric
  • IWaitable
  • IEvent (see below)

IEvent Methods

  • Set
  • Pulse
  • Reset
  • IsSet
