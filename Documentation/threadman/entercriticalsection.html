EnterCriticalSection

Definition

void EnterCriticalSection(void);

Description

This function temporarily disables thread preemption until ExitCriticalSection is called by raising the current interrupt processor priority (IRQL) to the level of the preemption interrupt. As a general rule, this should never be called by user code unless absolutely necessary. It is almost always better to use a synchronization primitive such as a Semaphore. Critical sections may be nested within one another.
